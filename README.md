# RimWeb — 웹 기반 콜로니 시뮬레이터

림월드(RimWorld)에서 영감을 받은 웹 콜로니 시뮬레이션 토이프로젝트.
종합설계프로젝트(캡스톤 디자인) 연습용.

## 실행

```bash
npm install
npm run dev   # http://localhost:5173
```

## 조작

| 입력 | 동작 |
| --- | --- |
| 좌클릭 드래그 | 선택한 도구를 영역에 적용 (벌목/채굴/벽/침대/비축구역/경작지/취소) |
| 좌클릭 (선택 도구) | 정착민 선택 → 우측 하단에 욕구 패널 표시 |
| 우클릭 (정착민 선택 중) | 직접 명령: 이동 / 벌목 / 채굴 / 건설 / 수확 / 운반 |
| 우클릭·휠클릭 드래그 | 화면 이동 |
| 마우스 휠 | 줌 |
| Space | 일시정지 / 재개 |
| 1 · 2 · 3 | 배속 (1x / 3x / 6x) |

## 게임 루프

정착민 3명이 우선순위에 따라 자율적으로 행동한다:

1. **생존**: 배고프면 식량(아이템 → 덤불 열매 순)을 찾아 먹고, 밤이거나 탈진하면 잠 (침대가 있으면 침대, 없으면 바닥)
2. **자재 배달**: 청사진(벽·침대)에 필요한 목재 운반
3. **건설**: 자재가 갖춰진 청사진 시공
4. **벌목·채굴**: 플레이어가 지정한 나무/바위 작업 → 목재/석재 드랍
5. **농사**: 다 자란 작물 수확(식량 드랍) → 빈 경작지에 파종 (수확 후 자동 재파종 루프)
6. **운반**: 땅에 떨어진 아이템을 비축구역으로
7. 할 일이 없으면 배회

덤불은 시간이 지나면 다시 자라고, 하루 4분(1배속 기준)의 낮/밤 주기가 있다.
정착민을 선택하고 우클릭하면 작업 우선순위를 무시하고 즉시 지시할 수 있다.
우상단 💾/📂 버튼으로 게임 전체를 localStorage에 저장/복원.

## 구조

```
src/
  constants.ts  게임 밸런스 수치 (욕구 감소율, 건설 비용, 하루 길이 등)
  types.ts      공유 타입/enum
  map.ts        타일맵 (지형·식물·바위·구조물·청사진·아이템) + 절차적 맵 생성
  astar.ts      8방향 A* 길찾기 + BFS 최근접 탐색
  pawn.ts       정착민: 욕구(포만감·휴식), 경로 추적 이동
  jobs.ts       작업 시스템: Job 상태기계들 + 우선순위 기반 잡 기버
  game.ts       시뮬레이션 틱, 배속, 낮/밤 주기, 타일 예약(이중 배정 방지)
  renderer.ts   PixiJS 렌더링 (타일/아이템/오버레이/정착민/밤 효과)
  camera.ts     팬/줌, 화면↔타일 좌표 변환
  input.ts      마우스/키보드 입력, 도구 적용
  ui.ts         DOM HUD (자원, 시계, 배속, 도구, 정착민 패널)
  state.ts      UI 공유 상태 (선택된 도구/정착민)
  main.ts       부트스트랩
```

그래픽은 의도적으로 도형/색상만 사용 (스프라이트는 추후 교체 가능).
