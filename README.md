# RimWeb — 웹 기반 콜로니 시뮬레이터

림월드(RimWorld)에서 영감을 받은 웹 콜로니 시뮬레이션 토이프로젝트.
종합설계프로젝트(캡스톤 디자인) 연습용.

## 실행

```bash
npm install
npm run dev    # http://localhost:5173
npm test       # 유닛 테스트 (Vitest)
npm run build  # 타입 체크 + 프로덕션 빌드
```

## 조작

| 입력 | 동작 |
| --- | --- |
| 좌클릭 드래그 | 선택한 도구를 영역에 적용 (벌목/채굴/벽/침대/비축구역/경작지/취소) |
| 좌클릭 (선택 도구) | 정착민 선택 → 우측 하단에 욕구 패널 표시 |

## 게임 루프

정착민 3명이 우선순위에 따라 자율적으로 행동한다:

1. **생존**: 배고프면 식량(아이템 → 덤불 열매 순)을 찾아 먹고, 밤이거나 탈진하면 잠 (침대가 있으면 침대, 없으면 바닥)
2. **자재 배달**: 청사진(벽·침대)에 필요한 목재 운반
3. **건설**: 자재가 갖춰진 청사진 시공
4. **벌목·채굴**: 플레이어가 지정한 나무/바위 작업 → 목재/석재 드랍
5. **농사**: 다 자란 작물 수확(식량 드랍) → 빈 경작지에 파종 (수확 후 자동 재파종 루프)
6. **운반**: 땅에 떨어진 아이템을 비축구역으로
7. 할 일이 없으면 배회

덤불은 시간이 지나면 다시 자라고, 하루 4분(1배속 기준)의 낮/밤 주기가 있다.
정착민을 선택하고 우클릭하면 작업 우선순위를 무시하고 즉시 지시할 수 있다.
우상단 📋 버튼으로 작업 우선순위 표를 열어 정착민×작업종류별로
1(높음)~4(낮음) 또는 "안 함"을 설정할 수 있다 (생존 욕구는 항상 최우선).
우상단 💾/📂 버튼으로 게임 전체를 localStorage에 저장/복원.

## 습격과 전투

2일차쯤부터 주기적으로 약탈자가 맵 가장자리에서 습격해 온다 (일수에 따라 규모 증가).

- **정착민**: 사거리 7타일 내 약탈자를 자동 사격 (벽·바위에 시야가 막히면 불가).
  R로 징집하면 하던 일을 멈추고 우클릭 위치로 이동시켜 배치할 수 있다.
- **약탈자**: 가장 가까운 정착민에게 근접 공격. 길이 벽으로 막혀 있으면
  가장 가까운 벽(체력 100)을 부수고 진입한다. 약 40%는 **원거리 무장**으로,
  5타일 거리를 유지하며 사격한다 (몸에 총신 표시).
- **전리품**: 원거리 약탈자는 죽으면 소총을 떨어뜨린다. 빈손 정착민이 자동으로
  주워 장착하며(이름 옆 🔫), 소총은 사거리 7→9, 피해 12→18로 강화된다.
- **터렛**(목재 8): 사거리 8타일 내 약탈자를 자동 사격하는 방어 시설. 통행을 막고,
  길이 막힌 약탈자는 벽처럼 터렛도 부순다.
- 체력이 0이 되면 약탈자는 사망, 정착민은 대개 쓰러졌다가 회복하지만 **20% 확률로
  사망**한다. 죽은 정착민은 시신과 유품(무기 등)을 남기고, 콜로니 전체가 하루 동안
  애도한다(기분 -12). 방치된 시신도 사기를 갉아먹으니 **무덤**(자재 불필요, 파기만)을
  만들어 매장할 것 — 매장은 운반 작업이 맡는다.
  쓰러진 동료는 다른 정착민이 **침대로 업어 나르고**, 침대 요양은 회복이 3배 빠르다.
- 모든 정착민이 쓰러지면 약탈자는 맵 밖으로 떠난다.

## 요리와 인구

**화덕**(목재 4)을 지으면 요리 담당 정착민이 생식량 2개를 모아 요리 1개를 만든다
(1인당 2개까지 비축). 정착민은 요리를 가장 선호하며, 따뜻한 식사는 기분을 올리고
생식(열매·생고기)은 기분을 떨어뜨린다.

며칠에 한 번 **방랑자가 합류**해 콜로니가 커진다 (최대 6명). 새 정착민은 작업
우선순위 표에 자동으로 추가된다.

## 야생동물과 사냥

사슴이 맵을 한가로이 돌아다니며, 줄어들면 하루에 한 마리씩 가장자리에서 보충된다.
드래그로 선택해 **사냥 지정**(주황 링)하거나 정착민 선택 후 우클릭으로 직접 사냥을
명령하면, 사냥 담당 정착민이 다가가 사격한다. 공격받은 사슴은 반대편으로 도망치고,
잡으면 고기(식량 6)를 드랍한다 — 겨울나기 비상 식량원.

## 날씨

가끔 비가 내린다 (겨울엔 눈). 비는 불을 빠르게 꺼 주고 작물을 더 빨리 자라게
하지만, 모두의 이동이 느려지고 정착민 기분이 살짝 나빠진다.

## 번개와 화재

가끔 번개가 떨어져 가연물(나무·작물·덤불·목조 벽·침대·아이템)에 불이 붙는다.
불은 인접 가연물로 번지고, 다 타면 연료가 소실되며 땅이 그을린다(흙으로 변함).
정착민은 **모든 일을 제쳐두고** 불을 끄러 달려가며, 불 위에 서 있는 모든 생명체는
피해를 입는다. 콜로니를 나무 벽으로만 두르면... 각오할 것.

## 기분과 멘탈 브레이크

정착민은 상황 요인(굶주림, 피로, 부상, 침대 부족, 습격 공포, 포만감 등)이 만드는
목표치를 향해 기분이 서서히 변한다. 기분이 임계 이하로 일정 시간 지속되면
**멘탈 브레이크** — 일을 내팽개치고 정처 없이 배회하며, 명령도 징집도 듣지 않는다.
발산이 끝나면 다소 후련해진 채 복귀한다. 요인 목록은 정착민 패널에서 확인.

## 계절

1년 = 봄·여름·가을·겨울 각 3일. 작물과 덤불은 여름에 빨리(1.2배), 가을에 천천히(0.7배)
자라고 **겨울에는 전혀 자라지 않는다** — 겨울나기 식량을 미리 비축할 것.

## 구조

```
src/
  animals.ts    야생동물: 배회/도주 AI, 스폰
  combat.ts     전투: 시야 체크, 정착민 자동 사격, 약탈자 AI (추격·벽 파괴·퇴각)
  constants.ts  게임 밸런스 수치 (욕구 감소율, 건설 비용, 전투 수치, 하루 길이 등)
  types.ts      공유 타입/enum
  map.ts        타일맵 (지형·식물·바위·구조물·청사진·아이템) + 절차적 맵 생성
  astar.ts      8방향 A* 길찾기 + BFS 최근접 탐색
  pawn.ts       정착민: 욕구(포만감·휴식), 경로 추적 이동
  jobs.ts       작업 시스템: Job 상태기계들 + 우선순위 기반 잡 기버
  game.ts       시뮬레이션 틱, 배속, 낮/밤 주기, 타일 예약(이중 배정 방지)
  renderer.ts   PixiJS 렌더링 (타일/아이템/오버레이/정착민/밤 효과)
  selection.ts  선택 로직: 박스 다중 선택, 일괄 징집/지정, 우클릭 명령 분배
  camera.ts     팬/줌, 화면↔타일 좌표 변환
  input.ts      마우스/키보드 입력, 도구 적용
  ui.ts         DOM HUD (자원, 시계, 배속, 도구, 정착민 패널)
  state.ts      UI 공유 상태 (선택된 도구/정착민)
  main.ts       부트스트랩
```

그래픽은 의도적으로 도형/색상만 사용 (스프라이트는 추후 교체 가능).

게임 로직(맵·길찾기·잡 기버·전투·세이브)은 렌더링(PixiJS)과 분리되어 있어
브라우저 없이 Node에서 그대로 테스트된다 — `src/__tests__/`에 A* 우회/봉쇄,
아이템 드랍 병합, 작업 우선순위/예약, 배달→시공 통합, 파종→수확 통합,
시야/벽 파괴/다운 회복, 세이브 라운드트립 등 32개 테스트.
